Regular_based_Superpixel_TMM_2014
=================================

Matlab code for Topology Preserved Regular Superpixel

Please cite the fllow papers if you used or was inspired by this code/work:

[1] Huazhu Fu, Xiaochun Cao, Dai Tang, Yahong Han, Dong Xu, "Regularity Preserved Superpixels and Supervoxels", IEEE Transactions on Multimedia (T-MM), vol. 16, no. 4, pp. 1165-1175, 2014.
[2] Dai Tang, Huazhu Fu, Xiaochun Cao, "Topology preserved regular superpixel", in IEEE International Conference on Multimedia & Expo (ICME), 2012, pp. 765-768.

This code is a primitive MATLAB code, which is simpler than that we used in the paper.
If you find any bugs, please contact Huazhu Fu (huazhufu@gmail.com).
